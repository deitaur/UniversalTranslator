# Universal Translator: Бизнес-описание продукта

## 🎯 Описание проекта
**Universal Translator** — это не просто десктопный переводчик, а комплексный AI-ассистент для Windows, предназначенный для преодоления языковых барьеров, улучшения качества коммуникации и обучения иностранным языкам. 

Приложение работает в фоновом режиме (через системный трей) и обеспечивает мгновенный доступ к переводам, улучшению текста и голосовому вводу в любом приложении (браузеры, мессенджеры, текстовые редакторы, IDE) с помощью глобальных горячих клавиш.

---

## 🚀 Ключевой функционал

### 1. Умный контекстный перевод
Приложение позволяет переводить выделенный текст или содержимое буфера обмена, не переключаясь между окнами.
* **In-place замена (Ctrl+Alt+R)**: Выделяете текст в любом окне, нажимаете шорткат, и программа автоматически заменяет оригинальный текст на переведенный (через эмуляцию ввода клавиатуры или буфер обмена).
* **Перевод во всплывающем окне (Ctrl+Alt+T)**: Показывает поверх всех окон элегантный попап с переведенным текстом.
* **Перевод буфера обмена (Ctrl+Alt+Y)**: Фоновый перевод содержимого буфера обмена.
* **Мульти-движки**: Поддержка DeepL (с отслеживанием лимитов API), Google Translate (бесплатный) и Yandex Translate.

### 2. Локальный искусственный интеллект (Ollama LLM)
Интеграция с локальными нейросетями (например, `qwen2.5:14b` через Ollama) обеспечивает конфиденциальность данных и нулевую стоимость запросов.
* **ИИ-роли и RAG**: Платформа поддерживает создание кастомных ИИ-ролей со своими системными промптами.
* **База знаний (RAG Engine)**: К каждой роли можно прикрепить папку с документами (txt, md, pdf и т.д.). Система автоматически разбивает их на чанки и использует алгоритм TF-IDF для векторного поиска, подставляя нужный контекст в диалог с ИИ.
* **Автоматическое сжатие контекста**: При заполнении лимита токенов ИИ автоматически делает саммари переписки, чтобы продолжить диалог без потери основной сути.

### 3. Встроенные ИИ-ассистенты (Роли)
* 💼 **Negotiator (Переговорщик) [Ctrl+Alt+N]**: 
  Помогает переписывать письма и сообщения, делая их более убедительными, профессиональными и уверенными. Анализирует тон текста и предлагает лучшие формулировки.
* 👨‍🏫 **English Teacher (Учитель английского) [Ctrl+Alt+E]**: 
  Интерактивный репетитор, использующий методики интервального повторения и активного вспоминания (Retrieval Practice). Помогает улучшать грамматику и словарный запас в формате живого общения.

### 4. Голосовой ввод (Speech-to-Text)
* **Интеграция Whisper [Ctrl+Alt+W]**: Запись голоса с микрофона и его мгновенная транскрибация или перевод в текст. Полезно для диктовки писем или перевода живой речи.

### 5. Пользовательский интерфейс и Настройки
* **Панель в трее (System Tray)**: Индикатор текущего движка перевода и визуальное отображение квот использования DeepL API.
* **Менеджер ролей**: Удобный графический интерфейс для создания, удаления и редактирования ИИ-ролей, с возможностью привязки папок базы знаний.
* **Модуль истории**: Сохранение сессий общения с ИИ для каждого чата. Менеджер сессий позволяет возвращаться к прошлым диалогам.

## 🏗 Архитектура и Технологии
Проект построен с использованием строгих принципов **Domain-Driven Design (DDD)** и **SOLID**, что делает его легко масштабируемым и надежным:

1. **`services/translators/`**: Инкапсулированная логика взаимодействия с внешними API (Google, Yandex, DeepL).
2. **`services/ai/`**: Изолированные модули для работы с LLM (Ollama), векторным поиском (RAG Engine) и аудиомоделями (Whisper).
3. **`storage/`**: Слой работы с файловой системой для хранения настроек, истории чатов и ИИ-ролей.
4. **`win32/`**: Платформозависимый слой, напрямую работающий с WinAPI для глобальных хуков клавиатуры, симуляции ввода и перехвата буфера обмена без конфликтов.
5. **`ui/`**: Пользовательский интерфейс, построенный на `customtkinter` для создания современного темного дизайна (Catppuccin Mocha theme).

## 💡 Целевая аудитория и ценность
* **IT-специалисты и фрилансеры**: Мгновенный перевод документации, переписка с иностранными заказчиками через "Переговорщика".
* **Изучающие языки**: Интерактивная практика языка без необходимости открывать браузер с ChatGPT.
* **Менеджеры и руководители**: Формирование грамотных, дипломатичных писем.
* **Бизнес-ценность**: Повышение продуктивности пользователя (за счет исключения переключений между окнами) и обеспечение приватности данных за счет локальных AI-моделей.
