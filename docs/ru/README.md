## Документация проекта RuEnDeeplGoogleTranslator

Добро пожаловать в проект RuEnDeeplGoogleTranslator. Это приложение предоставляет единый интерфейс для перевода текста с использованием нескольких передовых сервисов перевода, включая DeepL, Google Translate и Яндекс.

### 🚀 Обзор проекта

Основная цель этого приложения — оптимизировать процесс перевода путем интеграции ведущих API машинного перевода в единое, удобное для пользователя настольное приложение. Оно поддерживает перевод между русским (Ru) и английским (En), а также потенциально другими языками, используя сильные стороны различных сервисов для достижения оптимальной точности.

### ✨ Основные функции

*   **Поддержка нескольких переводчиков:** Бесшовная смена между переводчиками DeepL, Google и Яндекс.
*   **Удобный пользовательский интерфейс:** Чистый, интуитивно понятный графический интерфейс пользователя для простого ввода текста и просмотра переведенных результатов.
*   **Управление историей:** Отслеживает историю переводов для быстрого доступа к предыдущим переводам.
*   **Доступ на основе ролей:** (Если реализовано) Позволяет управлять различными ролями и разрешениями пользователей.
