# Universal Translator - Program Description

## Overview
The Universal Translator is a desktop application designed to streamline and enhance translation workflows across various platforms. It acts as a powerful utility by integrating multiple leading machine translation services (DeepL, Google, Yandex) directly into the operating system's clipboard and hotkey system.

## Key Functionalities
1.  **Global Hotkey Translation:** Users can translate selected text or the entire clipboard content instantly using dedicated hotkeys.
2.  **Contextual Chat Interface:** The application features a dedicated chat window for advanced conversational AI tasks. This allows users to interact with the AI in specific roles (e.g., Negotiator, Teacher) or summarize long dialogues to maintain context.
3.  **Session Management:** Users can save, load, and manage multiple chat sessions, ensuring that conversation history is preserved and easily retrievable.
4.  **AI Integration (Ollama):** Supports advanced context compression and dialogue continuation using local Large Language Models (LLMs) via Ollama.
5.  **Customization:** Provides granular control over the chat experience, including font family, font size, and the language used for chat interactions.

## Technical Highlights
*   **Multi-Engine Support:** Seamlessly switches between DeepL, Google, and Yandex translation engines based on user configuration.
*   **Clipboard Integration:** Robust handling of clipboard operations for reliable text capture and replacement.
*   **UI/UX:** Features a modern, customizable chat interface with selectable text bubbles and dedicated copy functionality.


# Универсальный переводчик - Описание программы

## Обзор
Universal Translator — это настольное приложение, предназначенное для оптимизации и улучшения рабочих процессов перевода на различных платформах. Он действует как мощная утилита, интегрируя несколько ведущих служб машинного перевода (DeepL, Google, Yandex) непосредственно в буфер обмена операционной системы и систему горячих клавиш.

## Ключевые функции
1. **Глобальный перевод с помощью горячих клавиш.** Пользователи могут мгновенно переводить выделенный текст или все содержимое буфера обмена, используя задаваемые горячие клавиши.
2. **Интерфейс контекстного чата.** В приложении имеется специальное окно чата для сложных диалоговых задач искусственного интеллекта. Это позволяет пользователям взаимодействовать с ИИ в определенных ролях (например, переговорщик, учитель) или суммировать длинные диалоги для поддержания контекста.
3. **Управление сеансами**. Пользователи могут сохранять, загружать и управлять несколькими сеансами чата, гарантируя сохранение истории разговоров и возможность ее легкого восстановления.
4. **Интеграция с искусственным интеллектом (Ollama):** поддерживает расширенное сжатие контекста и продолжение диалога с использованием локальных моделей большого языка (LLM) через Ollama.
5. **Настройка.** Обеспечивает детальный контроль над взаимодействием в чате, включая семейство шрифтов, размер шрифта и язык, используемый для взаимодействия в чате.

## Технические особенности
* **Поддержка нескольких механизмов:** плавно переключается между системами перевода DeepL, Google и Яндекс в зависимости от конфигурации пользователя.
* **Интеграция с буфером обмена:** Надежная обработка операций с буфером обмена для надежного захвата и замены текста.
* **UI/UX:** имеет современный настраиваемый интерфейс чата с возможностью выбора текстовых всплывающих окон и специальной функцией копирования.
